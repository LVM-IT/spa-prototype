# LVM Prototyp

## Prototyp starten

Nach dem Klonen des Repositories müssen die Abhängigkeiten mit `npm install` installiert werden.

Die Anwendung kann durch `npm run start` gestartet werden. Der Browser öffnet hierbei automatisch die App
